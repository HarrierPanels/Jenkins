## 2. Put sensitive data in credentials (github connection details, etc)
```
pipeline {
    agent any
    environment {
        GITHUB_TOKEN = credentials('github-token')
    }
    stages {
        stage('Example') {
            steps {
                sh "echo $GITHUB_TOKEN"
                // Use the credential as needed
            }
        }
    }
}
```
